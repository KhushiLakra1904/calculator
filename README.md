🧮 Clean Bootstrap Calculator
A sleek, responsive, and functional web-based calculator built with HTML5, JavaScript, and Bootstrap 5. This project was designed to demonstrate a modern UI layout without writing a single line of custom CSS.

🚀 Features
Zero Custom CSS: Entirely styled using Bootstrap 5 utility classes and components.

Responsive Design: The layout adjusts perfectly for desktop and mobile screens using the Bootstrap grid system.

Standard Operations: Supports addition, subtraction, multiplication, and division.

Error Handling: Built-in logic to handle invalid mathematical expressions.

Utility Keys: Includes "Clear" (C) and "Backspace" (←) functions for a better user experience.

🛠️ Built With
HTML5: Semantic structure.

Bootstrap 5: For the modern, professional UI (Grid, Cards, Buttons, and Shadow utilities).

JavaScript (ES6): Logic for calculations and display manipulation.
